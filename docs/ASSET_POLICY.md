# Asset Policy

## Google Drive only — private

Never commit:

- original portrait files;
- portrait private master;
- original diploma PDF;
- diploma private master;
- unredacted diploma;
- ZIP asset archives;
- private reports containing sensitive data.

## GitHub allowed — public production assets only

May be committed later after QA:

- web-ready portrait WebP;
- web-ready portrait fallback JPEG;
- redacted public diploma WebP;
- redacted public diploma fallback JPEG;
- approved logo files;
- approved public editorial images.

## Diploma requirement

Only the public redacted version may be used on the website.

The diploma number and other private identifiers must remain fully hidden.
