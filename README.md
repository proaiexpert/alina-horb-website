## Project workflow

1. Source assets are prepared and stored in private Google Drive.
2. Cowork creates the initial homepage implementation.
3. The result is reviewed.
4. Approved code is transferred into this repository.
5. Codex performs technical QA, responsive fixes and production preparation.

## Important

Private source assets, original diploma files and archives must never be committed to this repository.
